# SecurBlack Box 2.2.0

Локальное шифрование файлов без интернета и облаков. Скачайте EXE и пользуйтесь.

## Что нового в v2.2.0

### Критические исправления безопасности

- **Защита от переупорядочивания фрагментов** – теперь каждый блок имеет уникальный идентификатор, нельзя переставить или удалить часть файла незаметно
- **Проверка целостности всего файла** – SHA-256 хэш сохраняется и проверяется при расшифровке
- **Удалена передача паролей через командную строку** – пароли больше нельзя передавать через `-p`, только интерактивный ввод (безопаснее)
- **Очистка паролей из памяти** – пароли затираются сразу после использования
- **Безопасные временные файлы** – создаются с правами 600 (только текущий пользователь)
- **Защита от обхода через символические ссылки** – исправлен байпас политик Key Vault

### Конфиденциальность

- **Хэши файлов не логируются** – удалены из логов
- **Результаты верификации не пишутся в лог**

### Улучшения

- **Напоминание об обновлениях** – при 10-м запуске программа напоминает проверить GitHub
- **Обновлены ссылки** – официальный сайт: https://www.voltig.ru

### Совместимость

- Чтение файлов старого формата (v2) поддерживается
- При пересохранении файлы обновляются до формата v3

## Быстрый старт

Скачайте EXE, запустите, выберите файл и введите пароль.

```bash
# Зашифровать файл (пароль запросится интерактивно)
"SecurBlack Box 2.2.0.exe" -e документ.docx

# Расшифровать
"SecurBlack Box 2.2.0.exe" -d документ.docx.enc

# Зашифровать папку рекурсивно в 4 потока
"SecurBlack Box 2.2.0.exe" -e "C:\Документы" -r -t 4

# Сменить пароль
"SecurBlack Box 2.2.0.exe" -c файл.enc

# Проверить целостность
"SecurBlack Box 2.2.0.exe" -v файл.enc

# Создать управляемый ключ на флешке
"SecurBlack Box 2.2.0.exe" --create-key --key-root F:\ --key-label "USB-Key"

# Зашифровать управляемым ключом
"SecurBlack Box 2.2.0.exe" --encrypt-with-key документ.pdf --key-id <ID_ключа>

# Список ключей на подключенных дисках
"SecurBlack Box 2.2.0.exe" --list-keys
